# Next.js Project Setup and Basics

This repository serves as a comprehensive guide and reference for setting up a new Next.js project from scratch and understanding its core fundamentals.

You'll find step-by-step instructions for:
- Initializing a Next.js app (with both `create-next-app` and manual setup)
- Project structure explanation
- Key features like routing (App Router vs Pages Router), data fetching, styling, and deployment
- Best practices and common configurations

Perfect for beginners getting started with Next.js or as a quick reference for experienced developers spinning up new projects.

Built with Next.js 14+ and TypeScript.
